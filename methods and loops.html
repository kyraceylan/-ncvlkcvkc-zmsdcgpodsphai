Methods and Loops

Methods

A method is a function that is associated with an object. In JavaScript, methods are defined as properties of an object that are functions. They can be used to perform actions on the object or to return information about the object.

Loops

A loop is a programming construct that allows you to repeat a block of code multiple times. In JavaScript, there are several types of loops, including for loops, while loops, and do-while loops. Loops are useful for iterating over arrays, performing repetitive tasks, and controlling the flow of a program.

Switch Statements

A switch statement is a control flow statement that allows you to execute different blocks of code based on the value of a variable. In JavaScript, the switch statement evaluates an expression and matches its value against several case labels. If a match is found, the corresponding block of code is executed.
